# ESO Server Status Daemon

Приложение для Windows, которое показывает текущие статусы серверов The Elder Scrolls Online в системном трее.

## Что умеет
- Отслеживает 7 мегасерверов (PC EU/NA/PTS, Xbox EU/NA, PlayStation EU/NA).
- ЛКМ — открывает окно со сводкой статусов.
- Уведомляет, когда какой‑то сервер уходит в Offline или возвращается в Online (можно отключить).
- Может запускаться автоматически при входе в систему.

## Как установить и запустить
1) Скачайте и запустите `ESOServerStatusDaemon.exe`.
2) В системном трее появится иконка ESO. Если иконка не появилась — раскройте скрытые значки.

## Как пользоваться
- ЛКМ по иконке — открыть окно «ESO Server Status» со всеми серверами.
- ПКМ по иконке — меню:
  - «Включить уведомления» — включить/выключить всплывающие уведомления об изменениях.
  - «Автозагрузка» — включить/выключить запуск при входе в Windows.
  - «Выход» — закрыть приложение.

## Всплывающие уведомления
- При включённых уведомлениях приложение покажет список серверов, у которых изменился статус (Online/Offline).
- Если уведомления мешают, просто снимите галочку «Включить уведомления» в меню.

## Автозагрузка
- Включите пункт «Автозагрузка», чтобы приложение стартовало вместе с Windows.
- Отключить можно там же, в меню.

## Где настройки
- Настройки сохраняются автоматически здесь: `%AppData%/ESOServerStatusDaemon/settings.json`.
- Там хранится только включение/выключение уведомлений и автозагрузки.
